Testing Drag and Drop Functionality

Open the app at http://localhost:5174

Test Steps:

  1. Open the Dynamic UI Builder
  2. Drag and drop a few components (Table, Button, Text) to the canvas
  3. Try to drag and reorder the components within the canvas
  4. Verify that the components change their order when dragged

Expected Behavior:

  • Components should be draggable within the canvas
  • When dragging, the component should show a visual indication (reduced opacity)
  • Dropping a component in a new position should reorder the components
  • The order should persist during the session
